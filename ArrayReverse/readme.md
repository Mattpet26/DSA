# Daily Code Challenge - Array Reverse

## Perfect Sequence
*Author: Matthew Petersen*

---

### Problem Domain

Reverse a given array.

---

### Inputs and Expected Outputs

| Input | Expected Output |
| :----------- | :----------- |
| [1, 2, 3, 4, 5] | [5, 4, 3, 2, 1] |


---

### Big O


| Time | Space |
| :----------- | :----------- |
| O(n^2) | O(n) |

Utilizing a nested for-loop doubles our time complexity.
